# 🚀 Fast Gateway

# توجه ⚠️: این پروژه هنوز تکمیل نشده است
**Fast Gateway** یک تونل هوشمند و پنل مدیریت متمرکز VLESS over WebSocket است. این پروژه به عنوان **نسخه بهینه شده‌ی REN panel** توسعه یافته و به شما اجازه می‌دهد بدون نیاز به خرید سرور مجازی (VPS)، یک اتصال امن و پایدار را روی پلتفرم‌های ابری رایگان میزبانی کنید.

---

## ✨ ویژگی‌های نسخه بهینه شده (Fast Gateway)
* 🔹 **ظاهر کاملاً جدید:** تم رنگی آبی یکپارچه، مدرن و بهینه‌سازی شده برای حالت شب (Dark Mode).
* 🔹 **بدون نیاز به VPS:** امکان اجرای ۱۰۰٪ رایگان بر روی Render و Railway.
* 🔹 **سیستم خودکار Keep-Alive:** حل مشکل به خواب رفتن (Sleep) سرورهای رایگان با پینگ خودکار ۱۰ دقیقه‌ای.
* 🔹 **مدیریت پیشرفته کاربران:** امکان ساخت اتصال VLESS، تعیین سقف حجم مصرفی و کنترل وضعیت اتصال.
* 🔹 **بخش اختصاصی Clean IP:** امکان وارد کردن دامنه‌ها و آی‌پی‌های تمیز کلودفلر جهت کاهش پینگ و افزایش پایداری.

---

## ⚡ راهنمای راه‌اندازی سریع در Render

1. **فورک کردن پروژه:** ابتدا این ریپازیتوری را روی اکانت گیت‌هاب خود **Fork** کنید.
2. **ورود به Render:** به سایت [render.com](https://render.com) رفته و با اکانت گیت‌هاب خود لاگین کنید.
3. **ساخت سرویس جدید:** روی گزینه **New +** و سپس **Web Service** کلیک کنید.
4. **انتخاب ریپازیتوری:** مخزن فورک شده `Fast Gateway` را انتخاب کنید.
5. **تنظیمات دیپلوی:**
   * **Name:** `fast-gateway`
   * **Region:** `Frankfurt (Germany)` (برای پینگ بهتر به ایران)
   * **Build Command:** `pip install -r requirements.txt`
   * **Start Command:** `gunicorn main:app`
   * **Plan:** حتماً روی حالت **Free** قرار دهید.
6. **پایان کار:** روی دکمه **Deploy** کلیک کنید. پس از ۲ الی ۳ دقیقه، آدرس پنل اختصاصی شما آماده است.

---

## 🔒 ورود به پنل و استفاده

* **آدرس پنل شما:** `https://your-app-name.onrender.com/login`
* **رمز عبور پیش‌فرض:** `admin`

> ⚠️ **نکته امنیتی مهم:** بلافاصله پس از اولین ورود، به بخش **امنیت پنل** رفته و رمز عبور پیش‌فرض خود را تغییر دهید.

## 📱 اتصال به نرم‌افزارها
پس از ساخت اکانت در بخش *مدیریت اتصال‌ها*:
1. دکمه کپی کانفیگ را بزنید.
2. لینک کپی شده را در نرم‌افزارهای **v2rayNG** (اندروید)، **NekoBox** یا **v2rayN** (ویندوز) و **Streisand / FoXray** (آی‌او‌اس) وارد (Import) کنید.
3. متصل شوید و از اینترنت آزاد لذت ببرید!
